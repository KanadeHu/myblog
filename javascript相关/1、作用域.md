#### （1）关于eval(...) 欺骗词法作用域

首先我们看下面一段例子
```javascript
    function foo(str, a) {
        eval( str ); // 欺骗
        console.log( a, b 
    );}
    var b = 2;
    foo( "var b = 3;", 1 ); // 1, 3
```
eval()会在作用域内新生成一个词法内容
